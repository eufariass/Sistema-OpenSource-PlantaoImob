# Contribuindo

Obrigado por considerar contribuir com este projeto.

## Como contribuir

1. Faça um fork do repositorio.
2. Crie uma branch para sua alteracao (`feat/minha-feature` ou `fix/meu-ajuste`).
3. Faça commits pequenos e descritivos.
4. Abra um Pull Request com contexto claro do problema e da solucao.

## Padrao de qualidade

- Mantenha o estilo atual do projeto.
- Evite incluir credenciais ou dados sensiveis.
- Atualize a documentacao quando alterar comportamento.
- Teste localmente antes de abrir PR.

## Reportando bugs

Ao abrir issue, inclua:

- passos para reproduzir;
- comportamento esperado;
- comportamento atual;
- ambiente (local, Vercel, versao do Node, etc.).
